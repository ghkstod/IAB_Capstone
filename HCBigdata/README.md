<h1 align="left">
레포트 개요
</h1>
 <p>
   보건의료빅데이터 강의에서는 임상실험실습으로 연속혈당측정기를 착용하고, 실험군을 나누어 음료를 마신 뒤 또는 술을 마신 뒤 혈당,알코올,심박수 등을 측정한 데이터를 수집했다.
   이 데이터를 가지고 유의미한 결과를 보이는 데이터 분석, 시각화를 하는 것이 기말고사 레포트이다.
 </p>
